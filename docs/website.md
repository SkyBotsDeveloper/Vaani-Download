# Website Sync Notes

The Vaani website and this repository should stay aligned.

## Recommended Flow

1. Build the Windows installer from the private source repository.
2. Upload the installer to the latest GitHub Release in `Vaani-Download`.
3. Point the website download button to the GitHub Releases page or to the direct release asset URL.
4. Update release notes and changelog.

## Direct Download Option

Once a stable asset naming rule is fixed, the website can point directly to the latest asset URL from GitHub Releases.

# Installation Guide

## Windows

1. Open the latest release on the Vaani download repository.
2. Download the Windows installer file.
3. Run the installer.
4. Complete installation and launch Vaani.

## First Launch

Expected first-launch flow:

- no Google login requirement
- owner verification if enabled
- local settings and voice preferences

## Updates

New public installers should be uploaded to GitHub Releases so the website and GitHub repository both point to the same stable build.

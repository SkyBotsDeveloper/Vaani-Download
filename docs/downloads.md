# Downloads Policy

This repository is the public release channel for Vaani.

## Distribution Rules

- upload installers to GitHub Releases, not directly into the repository tree
- keep version numbers clear and stable
- use one canonical installer filename per release
- update release notes for each public build

## Recommended Release Asset

- `vaani-ai-1.2.3-setup.exe`

## Public vs Private Split

Public:

- releases
- docs
- changelog

Private:

- source code
- internal configs
- signing workflow
- build automation secrets
